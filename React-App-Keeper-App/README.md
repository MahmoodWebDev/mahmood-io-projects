- 💾 Local storage integration

## 🛠 Installation

1️⃣ Clone the repository:
\```bash
git clone https://github.com/mahmood-io-projects/React-Keeper-App.git
\```

2️⃣ Navigate to the project directory:
\```bash
cd React-Keeper-App
\```

3️⃣ Install the dependencies:
\```bash
npm install
\```

## 🚀 Usage

To run the app, execute the following command:

\```bash
npm start
\```

Open [http://localhost:3000](http://localhost:3000) in your browser to start keeping notes!
